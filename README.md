## 📋 Project Overview

This comprehensive data science project analyzes how Artificial Intelligence tools affect student academic performance, time management, and career confidence. Using real student data from 2026, we examine:

- 📈 **GPA Impact**: How AI tools affect academic grades
- ⏰ **Time Savings**: How many hours students save weekly
- 🎯 **Career Confidence**: How AI influences career preparedness
- 🤖 **Tool Comparison**: Which AI tools perform best
- 📚 **Major Differences**: Which fields benefit most from AI
- ⚖️ **Ethics Considerations**: How ethical concerns affect outcomes

## 📁 Dataset Information

### File: `AI_Impact_Student_Life_2026.csv`

**Size**: 1000+ student records

**Columns**:
| Column Name | Description | Data Type |
|-------------|-------------|-----------|
| Student_ID | Unique student identifier | String |
| Age | Student age (18-25) | Integer |
| Major | Academic discipline | String |
| Primary_AI_Tool | Main AI tool used | String |
| Task_Frequency_Daily | Number of AI tasks per day | Integer |
| Main_Usage_Case | Primary purpose for using AI | String |
| GPA_Baseline | GPA before AI adoption | Float |
| GPA_Post_AI | GPA after using AI | Float |
| Time_Saved_Hours_Weekly | Hours saved using AI | Integer |
| AI_Ethics_Concern | Level of ethical concern | String |
| Career_Confidence_Score | Confidence in career (1-10) | Integer |
